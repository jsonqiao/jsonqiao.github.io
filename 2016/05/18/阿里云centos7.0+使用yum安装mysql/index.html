阿里云centos7.0+使用yum安装mysql

发表于 2016-05-18   |   分类于 数据库   |  

参考链接

1.错误的安装姿势

1.1 安装命令
yum -y install mysql mysql-server mysql-devel
1.2 错误说明

以上命令在centos 6.5前能够正确安装的，但centos7.0+以后再使用以上命令安装时，则不能够成功安装mysql，而是安装了mysql的另一个分支名叫mariadb的数据库，关于此数据库各位可以自行谷歌之，抱歉本人讨厌百度。

2. 正确的安装姿势

上文已经对错误的安装原因进行了简要说明，这里开始上正确的干活，本人亲自操作过呢。

2.1 第一步，用如下命令更新软件包库
rpm -Uvh http://dev.mysql.com/get/mysql-community-release-el7-5.noarch.rpm
2.2 第二步，用如下命令检查仓库是否更新，即看包含mysql最新包

yum repolist enabled | grep “mysql.*”

如果使用如上命令，能够在查询结果中，看到MySQL 5.6 Community Server字样，说明已经加入，可安心进行下文的第三步了。

否则，各位看官又要自行google之原因了，因为本人一路顺利同时才疏学浅，不知原因。

2.3 第三步，用如下命令安装mysql

yum -y install mysql-community-server

在命令行写入如上代码，敲下回车，端起你的茶杯，follow next，好了。

如有意外，抱歉本人未遇到，可留言一起解决，看不懂英文，这个我可真帮不了。
