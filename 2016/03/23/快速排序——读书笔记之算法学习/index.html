快速排序——读书笔记之算法学习

发表于 2016-03-23   |   分类于 算法   |  

毕业将要3年之余，技术上出现瓶颈，不止如何再进一步，在迷茫中翻看以前学习的数据结构，当作温习，特作此笔记。

算法思想：

快速排序的核心思想为分治策略，何为分治，就是将一个大规模的问题拆分为小规模的一样或者类似的问题处理。

算法说明

首先在需要排序的序列中选出一个基准(pivot)，一般选取序列中的第一个元素作为基准(pivot)，然后在余下的序列元素中从右向左找出序列中比该基准(pivot)小的第一元素，同时，从左向右找出该序列中第一个比该基准(pivot)大的元素，如果从右向左的元素下标和从左向右元素的下标不等，则交换这两个元素，然后重复以上步骤，直到从右向左遍历的下标等于从左向右遍历的下标，此时，该序列被划分已该基准为中间元素，左边小于等于元素，右边大于该元素的序列，然后递归进行此操作即可。

    public class QuickSort {

    public static final void sort(int[] array) {

    quickSort(array, 0, array.length - 1);

    }

    public static final void quickSort(int[] array, int first, int last) {

    if (first &lt; last) {
    int pos = partition(array, first, last);
    quickSort(array, first, pos - 1);
    quickSort(array, pos + 1, last);
}

    }

    public static final int partition(int[] array, int first, int last) {

    int pivotPos = first;
int pivot = array[first++];
while (first &lt; last) {
    while (array[last] &gt; pivot &amp;&amp; last &gt; first) {
        last--;
    }
    while (first &lt; last &amp;&amp; (array[first] &lt;= pivot)) {
        first++;
    }

    // 交换
    if (first &lt; last) {
        int minValue = array[first];
        array[first] = array[last];
        array[last] = minValue;
    }
}

// 大于基准元素
if (array[first] &lt;= pivot) {
    array[pivotPos] = array[first];
    array[first] = pivot;
    return first;
}
return first - 1;

    }
