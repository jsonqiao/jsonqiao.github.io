Java compile-time constant

发表于 2015-03-05   |   分类于 Java   |   
# Creating a reference to a class using ".class" doesn't automatically initialize the Class object.
# There are actually three steps in preparing a class for use
# 1\. Loading. which is performed by the class loader. This finds the bytecodes and creates a Class \
#    object from those bytecodes.
# 2\. Linking. The link phase verifies the bytecodes in the class, allocates storage for static field,\
#    and if necessary, resolves all references to other classes made by this class.
# 3\. Initialization. If there's a superclass, initialize that. Executing static initializers and static\
#    initialization blocks.
# Java compile-time constant.
# 在编译时，编译器会将编译时常量的调用全部用常量替换
# 例如在类ClassInitable.str为编译时常量，
# 在类ClassInitialization's main方法中进行如下的调用
# System.out.println(ClassInitable.str)
# 则该语句经过编译后将会被替换常量值，即ClassInitable.str的值
例子
package com.json.rtti;

/**
 * Created by jsonqiao on 2015/3/5
 */
public class ClassInitialization {

 // Creating a reference to a class using ".class" doesn't automatically initialize the Class object.
 // There are actually three steps in preparing a class for use
 // 1\. Loading. which is performed by the class loader. This finds the bytecodes and creates a Class \
 // object from those bytecodes.
 // 2\. Linking. The link phase verifies the bytecodes in the class, allocates storage for static \
 // field and if necessary, resolves all references to other classes made by this class.
 // 3\. Initialization. If there's a superclass, initialize that. Executing static initializers and \
 // static initialization blocks.

 // Java compile-time constant.
 // 在编译时，编译器会将编译时常量的调用全部用常量替换
 // 例如在类ClassInitable.str为编译时常量，
 // 在类ClassInitialization's main方法中进行如下的调用
 // System.out.println(ClassInitable.str)
 // 则该语句经过编译后将会被替换常量值，即ClassInitable.str的值
 public static void main(String[] args) throws Exception {
     Class initable = ClassInitable.class;
     System.out.println("After creating Initable ref");
     //该条语句经过编译后等价于System.out.println("Hello World");
     System.out.println(ClassInitable.str);
     // Does trigger initialization:
     System.out.println(ClassInitable.a);
     // Does trigger initialization:
     System.out.println(ClassInitable2.a);
     Class initable3 = Class.forName("com.json.rtti.ClassInitable3");
     System.out.println("After creating Initable3 ref");
   }
}
