vim 插件管理工具pathogen安装

发表于 2015-05-20   |   分类于 Linux   |  

最近想在ubuntu下使用vim做一些基础的开发工作，vim作为编辑器之神，总归是要使用的，工欲善其事必先利其器，所以今天就带来vim插件管理工具pathogen的安装教程

官方网址：https://github.com/tpope/vim-pathogen
安装

mkdir -p ~/.vim/autoload ~/.vim/bundle && \
curl -LSso ~/.vim/autoload/pathogen.vim https://tpo.pe/pathogen.vim

 

配置

编辑~/.vimrc文件，添加如下内容

execute pathogen#infect()
syntax on
filetype plugin indent on

 

插件安装位置

通过如上的步骤，pathogen已经安装完毕，后续安装vim插件到~/.vim/bundle目录即可
