总结

Java序列化机制按如下协议进行



1. 首先写入序列化协议头，包含一个魔数以及协议版本

2. 然后递归写入序列化类的类描述信息，这里是先写子类的类描述信息，然后写父类的类描述信息，类描述信息主要包含

a. 类名

b. 类序列化版本号

c. 类的一些表示信息

d. 类实例的字段数以及字段类型信息

3. 然后写入类实例的基础类型信息，此处是按照先父类后子类的顺序进行

4. 最后写入类实例引用信息，次过程重复1～3的步骤
